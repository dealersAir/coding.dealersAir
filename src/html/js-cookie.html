cookie js

Для чтения и записи куки, в JavaScript используется свойство document.cookie, которое содержит строку в специальном формате.

Запись cookie

Записываем в свойство document.cookie строку, с параметрами разделенными ;

cookie_key=cookie_value - ключ и значение куки.

path=/mypath - путь, по которому будет доступ к куке. Если не указать, по умолчанию принимается текущий путь и все дочерние страницы.
Как правило используется path=/ - кука будет доступна на всех страницах сайта.

domain=site.com - домен или поддомен domain=subdom.site.com, на котором будет доступна кука. По умолчанию принимается текущий домен. Можно указать специальную маску domain=.site.com, тогда кука будет доступна на основном домене и всех его поддоменах.

expires=Mon, 07 Jan 2019 09:03:11 GMT - дата и время истечения куки в формате GMT. Если не указать, то кука будет сессионная и будет удалена после закрытия браузера. 

document.cookie = 'cookie_key=cookie_value; path=/; expires=Mon, 07 Jan 2019 09:03:11 GMT';

Чтение cookie

Все куки записаны в свойстве document.cookie в виде строки. Эта строка состоит из пар key=value разделенных точкой с запятой и пробелом'; '

Если никакие cookie к странице не привязаны, то в свойстве document.cookie находится пустая строка.

Для того, чтоб найти нужные ключ=значение, можно разбить строку по '; ' на массив и найти нужный ключ перебирая массив, разбивая пары на подмассивы. 

var cookie_array = document.cookie.split('; ');
for (var i = 0; i < cookie_array.length; i++) {
   var pair_array = cookie_array[i].split('=');

   if (cookie_key == pair_array[0]) {
      cookie_value = pair_array[1];
      break;
   }
}

Удаление cookie

Для удаления куки достаточно установить дату в прошлом, значение (cookie_value) не важно и можно не указывать. 

document.cookie = 'cookie_key=; path=/; expires=Thu, 01 Jan 1970 00:00:00 GMT';
