Специальные символы

Чтоб превратить спецсимвол в обычный, его необходимо экранировать, ставим перед ним обратный слеш \

Создание регулярного выражения

Регулярное выражение состоит из шаблона (pattern) и необязательных флагов (flags). 

var reg = /pattern/;
var reg = /pattern/flags;

или 

var reg = new RegExp('pattern');
var reg = new RegExp('pattern', 'flags');
При создании регулярки через new RegExp(), символьные классы (\w, \d, \s и т.д.), необходимо экранировать. 
var reg = new RegExp('\\w*');
