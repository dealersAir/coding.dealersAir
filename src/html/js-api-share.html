js api share

Чтоб отловить событие, когда посетитель поделился ссылкой в социальных сетях, он же шаринг — будем использовать api. К сожелению, не все соцсети дают такую возможность. Из всех сетей, с которыми я работал на данный момент, только facebook и однокласники позволяют это сделать.

Facebook share callback

Нам понадобится создать приложение в кабинете для разработчиков developers.facebook.com/apps. Указывем в настройках домен, на котором будем использовать api и копируем api id.

Подключаем js sdk, API_ID - api id нашего приложения. 

window.fbAsyncInit = function () {
   FB.init({
      appId: 'API_ID',
      autoLogAppEvents: true,
      xfbml: true,
      version: 'v3.3'
   });
}

(function (d, s, id) {
   var js, fjs = d.getElementsByTagName(s)[0];
   if (d.getElementById(id)) { return; }
   js = d.createElement(s); js.id = id;
   js.src = "https://connect.facebook.net/en_US/sdk.js";
   fjs.parentNode.insertBefore(js, fjs);
}(document, 'script', 'facebook-jssdk'));

Для вызова окна поделиться — используем метод Fb.ui().
HREF — url страницы, которой нужно поделиться.
Когда посетитель сделает публикацию — метод вернет response не равен undefined. 

FB.ui({
   method: 'share',
   href: 'HREF',
}, function (response) {
   if (response != undefined) {
      console.log('shared'); // пост опубликован
   }
});

Для передачи параметров: тайтл, описание, изображение и т.п., используем opengraph мета теги.
